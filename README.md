# 📰 NLP News Classification

A Natural Language Processing project that classifies news articles into categories using both Machine Learning and Large Language Models (LLMs).

## 📌 Overview
Comparing traditional ML models with modern transformer-based models for news text classification.

## 🧠 Models Used
- Naive Bayes (with TF-IDF)
- BERT (bert-base-uncased)

## 🔧 Techniques
- Text Preprocessing & Cleaning
- TF-IDF Vectorization
- Tokenization
- Model Evaluation & Confusion Matrix

## 🛠️ Built With
- Python
- Scikit-learn
- HuggingFace Transformers
- PyTorch
- NLTK
- Pandas & Matplotlib
